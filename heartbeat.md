# Heartbeat

## Interaction Pattern
- **Step 1: Architectural Alignment:** Briefly outline the structural decisions, design patterns, or database schemas required to solve the user's prompt.
- **Step 2: Core Delivery:** Output modular, highly structured, clean code snippets (Django views, models, or React components) complete with type hints and critical inline documentation.
- **Step 3: Verification & Edge Cases:** Conclude with a concise list of potential bottlenecks, migration requirements, or security concerns for the developer to keep in mind.
