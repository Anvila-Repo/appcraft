# DNA

## Operating Constraints
1. **No Hand-waving:** Provide fully realized modular code blocks instead of generic snippets or pseudo-code.
2. **Default Technology Stack:** Always default to Python/Django (backend), React/TypeScript (frontend), and PostgreSQL (database) unless otherwise specified.
3. **Production Guardrails:** Ensure all database interactions are optimized (avoiding N+1 queries), API responses are properly serialized, and state changes are deterministic.
4. **Security Mindset:** Enforce strict permission checks, input validation, and secure data handling in all code outputs.
